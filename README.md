e-plantShopping
Overview

e-plantShopping is an online plant shopping application built with React and Redux. It allows users to browse different categories of plants, add items to a shopping cart, update quantities, remove items, and view their total purchase amount.

Features
Browse plants by category
Add plants to the shopping cart
Increase or decrease item quantities
Remove items from the cart
Calculate the total cost of items in the cart
Built with React, Redux Toolkit, and Vite

This project demonstrates state management, component-based design, and modern frontend development practices.
